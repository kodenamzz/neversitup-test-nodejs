## Getting Started

### Installation

1. Clone the repository:

```bash
   $ git clone https://github.com/kodenamzz/neversitup-test-nodejs.git
   cd neversitup-test-nodejs.git
```

2. Install dependencies:

```bash
$ npm install
```

3. Run tests to ensure everything is set up correctly:

```bash
   $  npm test
```
